# Dados do Projeto

Esta pasta contém apenas **configurações públicas e exemplos neutros**.

## Arquivos

- `catalogo_indicadores.json`: catálogo de séries do BCB/SGS usadas pelo agente.
- `configuracao_alertas.json`: regras simples de monitoramento.
- `historico_consultas.csv`: histórico fictício e genérico, sem dados pessoais.
- `contexto_analitico.json`: escopo e restrições do agente.

## Observação

As **leituras reais das séries** devem ser buscadas pela aplicação em tempo de execução, usando a API oficial do Banco Central.
